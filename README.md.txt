# Exercise 1: Add Data and Format

This exercise involved improving the readability and structure of an Excel worksheet by performing the following tasks:

- Deleted unnecessary columns and adjusted column widths.
- Corrected misaligned numbers and added missing country data.
- Applied formatting to headers and data for better presentation.
- Added a new worksheet for exchange rates and formatted it.

## Files
- **1-Add-Data-Begin.xlsx**: Initial file before applying changes.
- **1-Add-Data-Last.xlsx**: Final file after completing all tasks.

## Tools Used
- Microsoft Excel 2021
- Formatting and basic data manipulation techniques.
